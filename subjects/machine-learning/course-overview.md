## Course Objectives

By the end of this course, learners will be able to:

- Explain core Machine Learning concepts and paradigms
- Formulate learning problems mathematically
- Implement standard ML workflows
- Evaluate models and interpret results
- Apply ML responsibly in real-world contexts

---

## Course Structure

The course is organised into the following units:

- Unit 1: Introduction and Foundations of Machine Learning
- Unit 2: Supervised Learning
- Unit 3: Unsupervised Learning
- Unit 4: Model Evaluation and Improvement
- Unit 5: Practical Applications and Case Studies

Each unit is supported by:

- Lecture notes and presentations
- Practicals
- Microprojects
- Assessments
- Supplementary resources

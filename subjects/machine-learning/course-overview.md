---

## Course Objectives

By the end of this course, learners will be able to:

- Explain core Machine Learning concepts and paradigms
- Formulate learning problems mathematically
- Implement standard ML workflows
- Evaluate models and interpret results
- Apply ML responsibly in real-world contexts

---

## Course Structure

The course is organised into **5 units** following the **MSBTE K-Scheme Curriculum (Course 316316, Semester 6)**:

- **[Unit I: Introduction to Machine Learning](./units/unit-01-introduction-to-machine-learning.md)** - ML Basics, Types, Python for ML
- **[Unit II: Data Preprocessing](./units/unit-02-data-preprocessing.md)** - Data Cleaning, Missing Values, Dataset Splitting
- **[Unit III: Feature Selection](./units/unit-03-feature-selection.md)** - Feature Scaling, Selection Methods, Extraction
- **[Unit IV: Supervised Learning](./units/unit-04-supervised-learning.md)** - Classification, Regression, Model Evaluation
- **[Unit V: Unsupervised Learning](./units/unit-05-unsupervised-learning.md)** - Clustering, Dimensionality Reduction, PCA

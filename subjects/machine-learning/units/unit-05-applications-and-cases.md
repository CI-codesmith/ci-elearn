# Unit 5: Applications and Case Studies

---

## Unit Overview

This unit applies Machine Learning concepts to real-world problems
through practical applications and case studies.

---

## Learning Objectives

After completing this unit, learners will be able to:

- Identify suitable ML approaches for real problems
- Analyse real-world case studies
- Reflect on ethical and practical considerations

---

## Learning Components

This unit includes:

- Case-study-based lectures
- Applied practicals
- Microprojects:
  - [Microproject 5.1: Real-World Application](../microprojects/MICROPROJECT_5_1.md)
  - [Microproject 5.2: Case Study Analysis](../microprojects/MICROPROJECT_5_2.md)
  - [Microproject 5.3: Ethical Considerations](../microprojects/MICROPROJECT_5_3.md)
- Comprehensive assessments

# Unit 2: Supervised Learning

---

## Unit Overview

This unit introduces supervised learning, where models are trained using
labelled data to perform prediction and classification tasks.

---

## Learning Objectives

After completing this unit, learners will be able to:

- Explain the supervised learning paradigm
- Distinguish between regression and classification
- Understand common supervised learning workflows

---

## Learning Components

This unit includes:

- Lecture notes and presentations
- Practicals aligned with supervised learning concepts

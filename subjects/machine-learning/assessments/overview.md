This section defines the assessment philosophy and structure for the
Machine Learning subject under Chatake Innoworks eLearn.

The framework is designed to encourage continuous learning, conceptual
clarity, and applied understanding rather than rote memorisation.

---

## Assessment Philosophy

- Assessments are learning tools, not just evaluation tools
- Emphasis on reasoning, interpretation, and application
- Progressive difficulty aligned with unit advancement
- Clear separation between formative and summative assessments

---

## Assessment Components

The Machine Learning subject uses the following assessment types:

- Weekly Tests
- Class Tests
- Prelims (End-of-course assessment)

Each component serves a distinct pedagogical purpose and is documented
separately.

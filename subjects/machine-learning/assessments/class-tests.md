# Class Tests

Class tests are **periodic summative assessments** conducted after
multiple units have been completed.

---

## Purpose

- Evaluate cumulative understanding
- Test application of concepts
- Assess problem-solving ability

---

## Structure

- Medium duration (45–60 minutes)
- Mix of conceptual and applied questions
- Includes problem formulation and interpretation

---

## Coverage

- Typically spans 2–3 units
- Questions may integrate multiple concepts
- Balanced difficulty

---

## Evaluation

- Moderate weightage
- Graded formally
- Used for academic progress tracking

## Practical Objective

This practical teaches time series analysis and forecasting techniques. Students will learn:
- Time series patterns and components
- Autoregressive and moving average models
- Exponential smoothing
- ARIMA and SARIMA models
- Modern deep learning approaches for time series

## Prerequisites

- Completion of Practicals 1-11

---

## Topics Covered

1. **Time Series Fundamentals**
   - Trend, seasonality, noise components
   - Stationarity and differencing
   - Autocorrelation and partial autocorrelation (ACF/PACF)
   - Decomposition methods

2. **Classical Methods**
   - Moving averages
   - Exponential smoothing
   - Holt-Winters method
   - Simple and double exponential smoothing

3. **ARIMA Family**
   - AutoRegressive (AR) model
   - Moving Average (MA) model
   - ARIMA (AutoRegressive Integrated Moving Average)
   - SARIMA (Seasonal ARIMA)
   - Parameter selection (p, d, q)

4. **Advanced Statistical Methods**
   - Vector Autoregression (VAR)
   - Error correction models
   - GARCH for volatility forecasting

5. **Machine Learning Approaches**
   - Feature engineering for time series
   - Regression models on time lags
   - Ensemble methods for forecasting
   - LightGBM/XGBoost for time series

6. **Deep Learning Methods**
   - LSTM (Long Short-Term Memory)
   - GRU (Gated Recurrent Unit)
   - Attention mechanisms
   - Transformer-based models

7. **Evaluation and Validation**
   - Walk-forward validation
   - Time series cross-validation
   - Forecast accuracy metrics (MAPE, RMSE)
   - Residual analysis

## Learning Outcomes (Practical Level)

After completing this practical, students will be able to:

1. **Understand anomaly detection** concepts and applications
2. **Implement anomaly detection algorithms**
3. **Handle imbalanced anomalous data**
4. **Evaluate anomaly detection** models
5. **Apply anomaly detection** to real-world scenarios

---

## Practical Objective

This practical covers techniques for identifying unusual patterns in data. Students will learn:
- Statistical and probabilistic approaches
- Isolation and proximity-based methods
- Deep learning for anomaly detection
- Evaluation metrics for imbalanced problems
- Practical applications in fraud detection and system monitoring

---

## Prerequisites

- Completion of Practicals 1-12
- Understanding of statistical distributions
- Knowledge of clustering concepts

---

## Topics Covered

1. **Statistical Methods**
   - Z-score and modified Z-score
   - IQR (Interquartile Range) method
   - Gaussian distribution-based detection
   - Grubbs test and Dixon's test

2. **Distance-Based Methods**
   - K-Nearest Neighbors (KNN)
   - Local Outlier Factor (LOF)
   - Distance metrics in high dimensions

3. **Density-Based Methods**
   - DBSCAN clustering
   - Density-based outlier factors
   - Local density comparison

4. **Isolation Methods**
   - Isolation Forest
   - Random Forest-based anomaly detection
   - Tree partitioning strategy

5. **Probabilistic Methods**
   - Gaussian Mixture Models (GMM)
   - One-class SVM
   - Bayesian networks

6. **Time Series Anomalies**
   - Seasonal decomposition
   - Residual analysis
   - Moving average deviations
   - Prophet and STL decomposition

7. **Deep Learning Approaches**
   - Autoencoders for detection
   - Variational Autoencoders (VAE)
   - LSTM-based anomaly detection
   - Adversarial networks

8. **Evaluation Metrics**
   - Precision, Recall, F1-Score
   - ROC curves for imbalanced data
   - Isolation index
   - Interpretability of results

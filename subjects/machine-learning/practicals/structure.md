# Practical Structure

This document defines the standard structure for Machine Learning
practical exercises.

---

## Typical Components

A practical exercise may include:

- Problem statement and objective
- Description of input data
- Step-by-step implementation guidance
- Expected observations or outcomes
- Short reflection questions

---

## Level of Guidance

- Practicals are guided, not exploratory
- Clear instructions are provided
- Learners are encouraged to modify parameters and observe effects

---

## Tools and Environment

- Practicals may use Python and common ML libraries
- Emphasis is on understanding workflows, not tool mastery
- Environment setup is provided separately

# Practical 1: IDE Installation and ML Libraries Setup

## Overview
This practical focuses on setting up a complete Machine Learning development environment. You'll install Python, an IDE (Integrated Development Environment), and essential ML libraries to prepare for the course.

## Learning Objectives
- Install and configure Python 3.9+
- Set up a virtual environment
- Install core ML libraries (NumPy, Pandas, Matplotlib, Scikit-learn)
- Configure Jupyter Notebook
- Verify installations with test scripts

## Files in This Folder
- `Practical-1-Comprehensive.md` - Detailed step-by-step guide
- `Practical_1_Complete_Notebook.ipynb` - Jupyter notebook with examples
- `Jupyter-setup.ipynb` - Setup verification notebook
- `test_installation.py` - Python script to test installations
- `SUBMISSION_TEMPLATE_Practical_1.md` - Simplified submission template

## Quick Start
1. Follow the installation guide in `Practical-1-Comprehensive.md`
2. Run `test_installation.py` to verify your setup
3. Complete the submission template

## Requirements
- Python 3.9 or higher
- At least 4GB RAM
- Internet connection for library installations

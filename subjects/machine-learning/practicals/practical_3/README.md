## Course Information
- **Course Code:** 316316
- **Course Title:** Machine Learning
- **Semester:** 6th
- **Practical Duration:** 2 Hours
- **Learning Outcome Code:** LLO 3.1
- **Aligned Course Outcomes:** CO2

---

## Learning Outcomes (Practical Level)

After completing this practical, students will be able to:

1. **Read and parse different data formats** (CSV, JSON, XML, TXT)
2. **Understand file structure** and format specifications
3. **Handle various data formats** using appropriate libraries
4. **Process and validate imported data**
5. **Convert between different data formats**

---

## Practical Objective

This practical teaches students to work with different data sources and formats commonly used in real-world scenarios. Students will learn:
- How to read data from various file formats
- Libraries for handling different formats (Pandas, JSON, XML)
- Data validation after import
- Format conversion techniques

---

## Prerequisites

- Completion of Practical 1 & 2
- Knowledge of Pandas and basic Python

---

## Topics Covered

1. **CSV Files**
   - Reading CSV with Pandas
   - Handling delimiters
   - Dealing with headers and index

2. **JSON Files**
   - Reading JSON structures
   - Handling nested JSON
   - Converting JSON to DataFrame

3. **XML Files**
   - Parsing XML with ElementTree or lxml
   - Extracting data from XML
   - Converting XML to tabular format

4. **Text Files**
   - Reading text files
   - Parsing structured text data
   - Handling different encodings

5. **Data Validation**
   - Checking data integrity
   - Validating data types
   - Identifying inconsistencies

6. **Format Conversion**
   - Converting between formats
   - Saving data in different formats
   - Data interoperability
